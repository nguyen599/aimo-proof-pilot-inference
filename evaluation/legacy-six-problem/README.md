# Legacy six-problem evidence

This directory preserves inputs and results from the original six-problem AIMO
Proof Pilot experiment. It is historical evidence, not executable evaluation
code and not an input to the active IMO 2025 pipeline.

Contents:

- `problems.csv`: the six original markscheme problems.
- `results/`: committed DIVALL and ALTORO submissions and trace summaries.

The obsolete runner was removed when the repository was consolidated around the
single YAML-configured evaluator in the parent `evaluation/` directory.
