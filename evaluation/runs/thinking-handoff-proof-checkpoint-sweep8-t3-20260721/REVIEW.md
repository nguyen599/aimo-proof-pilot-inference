# Thinking-Budget Handoff Review

Review each prompt/temperature group for:

1. Fidelity to the saved reasoning, without invented lemmas.
2. Preservation of reusable equations, notation, and proved facts.
3. Precise description of failed approaches and the unresolved bottleneck.
4. Concrete next steps that help a fresh solver avoid repetition.
5. Concision and valid XML structure.

Select a default only when it has no severe fidelity failure and is useful on at least six of the eight selected cases.
