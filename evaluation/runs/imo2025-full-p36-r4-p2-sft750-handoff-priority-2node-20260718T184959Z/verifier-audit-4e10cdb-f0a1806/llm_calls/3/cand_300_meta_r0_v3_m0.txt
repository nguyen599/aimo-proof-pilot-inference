Let $\mathbb{N}$ denote the set of positive integers. A function $f\colon\mathbb{N}\to\mathbb{N}$ is said to be bonza if
\[
f(a)\quad\text{divides}\quad b^a-f(b)^{f(a)}

## Solution
We show that the smallest constant \(c\) is \(4\).

First, we derive basic properties of any bonza function \(f:\mathbb{N}\to\mathbb{N}\).

**Lemma 1.** \(f(1)=1\).  
*Proof.* For \(a=1\) the condition gives \(f(1)\mid b-f(b)^{f(1)}\) for all \(b\). Taking \(b=1\) we get \(f(1)\mid 1-f(1)^{f(1)}\). Since \(f(1)^{f(1)}\equiv0\pmod{f(1)}\), we have \(1\equiv0\pmod{f(1)}\), so \(f(1)=1\). ∎

**Lemma 2.** \(f(n)\mid n^n\) for every \(n\).  
*Proof.* Take \(a=n\) and \(b=n\): \(f(n)\mid n^n-f(n)^{f(n)}\). Because \(f(n)\mid f(n)^{f(n)}\), it follows that \(f(n)\mid n^n\). ∎

**Lemma 3.** \(f(2)\in\{1,2,4\}\).  
*Proof.* From \(a=2,b=2\) we get \(f(2)\mid 2^2-f(2)^{f(2)}=4-f(2)^{f(2)}\). Hence \(f(2)\mid4\). ∎

**Lemma 4.** \(f(3)\in\{1,3,9,27\}\).  
*Proof.* From \(a=3,b=3\) we get \(f(3)\mid27-f(3)^3\). Thus \(f(3)\) divides \(27\). The only positive divisors of \(27\) are \(1,3,9,27\). ∎

**Lemma 5.** The function \(f\) is determined by its values on powers of two. For every \(k\ge1\), \(f(2^k)\) is a power of two.  
*Proof.* Using \(a=2^k,b=2\) we obtain \(f(2^k)\mid 2^{2^k}-f(2)^{f(2^k)}\). If \(f(2)=2\) or \(4\), then \(f(2)^{f(2^k)}\) is a power of two, so the difference is a power of two. Consequently every odd divisor of \(f(2^k)\) would have to divide a power of two, forcing it to be \(1\). Thus \(f(2^k)=2^{e_k}\) for some \(e_k\). ∎
