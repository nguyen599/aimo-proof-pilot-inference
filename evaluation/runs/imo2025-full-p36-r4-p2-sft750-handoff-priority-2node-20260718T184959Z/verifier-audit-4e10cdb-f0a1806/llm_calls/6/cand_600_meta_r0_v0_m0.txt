## Solution
Let \(n = 2025\).  The grid consists of \(n \times n\) unit squares.  A placement of rectangular tiles means that the tiles are axis‑aligned and cover a subset of the squares, each square being covered by at most one tile.  The condition that each row and each column contains exactly one uncovered square implies that the uncovered squares form a permutation matrix: there is a permutation \(\sigma\) of \(\{1,\dots,n\}\) such that the uncovered square in row \(i\) is at column \(\sigma(i)\).

Because the problem is symmetric under arbitrary permutations of rows and columns (the grid geometry and the covering condition are preserved, and the number of tiles needed is invariant), we may apply a row permutation and a column permutation that transform the uncovered squares into the main diagonal.  Hence we may assume without loss of generality that the uncovered squares are exactly the cells \((i,i)\) for \(i = 1,\dots,n\).  The covered squares are therefore all cells with \(i \neq j\).

We need the minimum possible number of tiles that partition the set of off‑diagonal cells.

---

### Upper bound: a construction with \(2n-2\) tiles

For \(i = 2,\dots,n\) define  
\[
R_i = \{(i,j): 1 \le j \le i-1\},
\]

For \(j = 2,\dots,n\) define  
\[
C_j = \{(i,j): 1 \le i \le j-1\},
\]  
which is a vertical rectangle (columns \(j\), rows \(1\) to \(j-1\)).  These rectangles are disjoint and cover all cells with \(i < j\).  
The collection \(\{R_i\}_{i=2}^n \cup \{C_j\}_{j=2}^n\) consists of \(2n-2\) tiles and satisfies the condition.  Hence the minimum number of tiles is at most \(2n-2 = 2\cdot2025-2 = 4048\).
